颜色主题让您可以通过更改进度环、图表和小组件使用的颜色来自定义 Steps 的外观。

<Note>颜色主题是 Pro 功能。免费用户使用默认的经典主题。</Note>

Steps 包含 7 种内置颜色主题。每种主题有两种颜色：一种用于进行中状态，一种用于目标已完成状态。

| 主题 | 进行中 | 已完成 |
|-------|-------------|-----------|
| **经典** | 橙色 | 绿色 |
| **石板** | 石板蓝 | 蓝色 |
| **珊瑚** | 珊瑚色 | 玫瑰色 |
| **森林** | 琥珀色 | 翡翠绿 |
| **暮光** | 紫罗兰 | 靛蓝 |
| **青色** | 青色 | 青蓝色 |
| **腮红** | 粉色 | 紫色 |

## 自定义颜色

使用自定义主题选项选择您自己的颜色。您可以分别设置：

- **进行中** - 在您努力达成目标时显示的颜色
- **已完成** - 达成目标后显示的颜色

使用 iOS 颜色选择器选择您喜欢的任何颜色。

## 主题应用范围

您选择的主题颜色会出现在整个应用中：

- 主页仪表盘上的进度环
- 图表条形和高亮
- 主屏幕和锁屏小组件
- 连续打卡徽章装饰

<Tip>更改主题会自动更新您的小组件以匹配。</Tip>

## 如何更改主题

1. 打开**设置**
2. 点击**颜色主题**
3. 选择预设主题或点击**自定义**以选择您自己的颜色
