| 样式 | 描述 |
|-------|-------------|
| **透明** | 锻炼统计数据显示在透明背景上，可叠加到任意图像 |
| **叠加** | 统计数据直接叠加在路线地图上 |
| **边框** | 路线地图带有整洁的边框统计数据框架 |
| **照片** | 统计数据叠加在您从相册选择的照片上 |

<Tip>叠加和边框样式适用于有 GPS 路线的户外锻炼。</Tip>

## 宽高比

为您的分享目标选择最佳比例：

| 比例 | 最适合 |
|-------|----------|
| **16:9** | YouTube、横向帖子 |
| **4:3** | 演示文稿、横向照片 |
| **1:1** | Instagram 动态、头像 |
| **3:4** | 竖向照片 |
| **9:16** | Instagram 故事、TikTok、Reels |

## 地图选项

对于叠加和边框样式，可自定义地图外观：

- **地图类型** - 标准或卫星
- **外观** - 浅色或深色
- **兴趣点** - 全部显示或全部隐藏（仅限标准地图）
- **海拔样式** - 平面或写实（仅限卫星地图）

## 保存到相册

<Note>保存到相册是 Pro 功能。</Note>

点击保存按钮将您的样式锻炼图像作为 PNG 直接导出到您的相册。

## GPX 导出

<Note>GPX 导出是 Pro 功能。</Note>

将锻炼路线导出为 `.gpx` 文件，用于其他健身应用、地图工具或路线规划器。GPX 文件包含带时间戳的完整 GPS 轨迹。
