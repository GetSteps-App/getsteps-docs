设置页面让您可以自定义 Steps 的外观和功能。通过主页仪表盘上的齿轮图标进入。

| 设置 | 选项 | 备注 |
|---------|---------|-------|
| **外观** | 跟随系统、浅色、深色 | 跟随设备或手动设置 |
| **颜色主题** | 7 种预设 + 自定义 | Pro 功能。参见[颜色主题](/features/color-themes) |
| **单位制** | 公制、英制 | 影响距离显示（公里或英里） |
| **每周起始日** | 周日、周一 | 更改日历和每周视图 |
| **步数统计方式** | 天、周、月 | 周和月是 Pro 功能 |
| **每日目标** | 1,000 - 20,000 | 参见[每日目标](/features/daily-goal) |
| **应用锁定** | 开、关 | Pro 功能。参见[应用锁定](/features/app-lock) |
| **通知** | 开、关 | Pro 功能。参见[通知](/features/notifications) |
| **主页洞察** | 关闭、信息、平均值 | 步数下方显示的内容 |
| **显示天气** | 开、关 | 显示温度和湿度 |
| **应用图标** | 17 款图标 | 参见[应用图标](/features/app-icon-selector) |
| **主屏幕小组件** | 设置指南 | 添加小组件的说明 |

## 语言

Steps 跟随您设备的语言设置。要更改应用语言：

1. 打开 iPhone 的**设置**
2. 在应用列表中找到 **Steps**
3. 更改**语言**设置

## 支持

- **反馈面板** - 提交功能请求并对建议投票
- **写下评价** - 在 App Store 上为 Steps 评分
- **重新开始引导设置** - 再次完成设置流程
- **重新同步 Apple Health 数据** - 刷新您的活动数据
- **管理订阅** - 更改或取消 Pro 计划（仅限 Pro 用户）
- **服务条款** - [getsteps.app/terms](https://getsteps.app/terms)
- **隐私政策** - [getsteps.app/privacy](https://getsteps.app/privacy)
