Steps 提供多种小组件，让您无需打开应用即可查看步数、连续打卡和锻炼数据。

| 小组件 | 尺寸 | 描述 |
|--------|-------|-------------|
| **今日步数** | 小号、锁屏（矩形、内联、圆形） | 当前步数和目标进度 |
| **月度日历** | 中号 | 显示当月每日目标完成情况的日历网格 |
| **年度日历** | 中号、大号 | 带目标完成热图的全年日历 |
| **年度目标** | 小号、中号 | 实现年度步数目标的进度，含状态和统计数据 |
| **每小时图表** | 中号 | 今日按小时步数条形图 |
| **每周图表** | 中号 | 过去 7 天步数条形图 |
| **每月图表** | 中号 | 当月步数条形图 |
| **连续打卡** | 小号、中号 | 当前连续打卡天数和每周目标进度 |
| **锻炼亮点** | 小号 | 最近一次锻炼的统计数据 |
| **步数排行榜排名** | 小号 | 排行榜今日前 5 名以及您的排名 |
| **开始锻炼** | 小号、中号 | 一键直接进入某个锻炼类型 |

## 步数排行榜排名小组件

步数排行榜排名小组件将您的[步数排行榜](/features/stepboard)放到主屏幕上。它显示今日前 5 名步行者，领先者带有奖牌表情符号，如果您不在前 5 名之内，还会显示您自己的排名。添加该小组件时，您可以选择它显示哪个榜单——World 榜单或您加入的任意群组。点击小组件即可直接跳转到该榜单。

## 开始锻炼小组件

{/* TODO: Screenshot of the Start Workout widget on the Home Screen */}
<Frame caption="一键开始锻炼的「开始锻炼」小组件">
  <img src="/images/screenshots/widgets-start-workout.jpg" alt="Start Workout widget" />
</Frame>

开始锻炼小组件让您可以直接从主屏幕开始追踪[锻炼](/features/workout-session)。点击它，Steps 会直接打开该锻炼类型——无需翻找菜单。

- **小号**——一个可配置的锻炼类型，带有开始按钮
- **中号**——您的主要锻炼，外加另外三种类型的快速启动快捷方式

添加小组件时，请选择它要开始的锻炼类型。如果已有锻炼正在进行，点击小组件会改为打开进行中的锻炼。

## 年度目标小组件

年度目标小组件追踪您实现[年度步数目标](/features/yearly-goal)的进度。它显示进度条、您是按计划进行还是落后，以及辅助统计数据。提供小号和中号尺寸。

## Live Activity

进行中的锻炼期间，Live Activity 会出现在您的锁屏和 Dynamic Island 上（iOS 26+）。显示内容包括：

- 锻炼类型图标
- 已用时间
- 覆盖距离
- 消耗卡路里
- 当前配速

Dynamic Island 每 30 秒循环显示各项数据。

## 添加小组件

<Steps>
  <Step title="长按主屏幕">
    触摸并按住主屏幕上的空白区域，直到应用开始抖动。
  </Step>
  <Step title="点击 + 按钮">
    点击左上角的 **+** 按钮打开小组件库。
  </Step>
  <Step title="搜索 Steps">
    在搜索栏中输入"Steps"以查找所有可用小组件。
  </Step>
  <Step title="选择小组件">
    选择您想要的小组件类型和尺寸，然后点击**添加小组件**。
  </Step>
</Steps>

今日步数小组件也可用于锁屏，提供三种样式：

- **矩形** - 显示步数和进度条
- **内联** - 日期行中的紧凑步数显示
- **圆形** - 圆形仪表中的步数

<Tip>您的小组件会自动更新最新步数数据，并反映您选择的颜色主题。</Tip>

## 保持小组件实时更新

为了让小组件能够频繁刷新显示最新步数数据，您需要为 Steps 启用**后台应用刷新**。

<Steps>
  <Step title="打开设置">
    前往**设置 → 通用 → 后台应用刷新**。
  </Step>
  <Step title="启用后台应用刷新">
    确保主开关设置为 **Wi-Fi 与蜂窝数据**（或至少为 **Wi-Fi**）。
  </Step>
  <Step title="为 Steps 启用">
    向下滚动，确保 **Steps** 旁边的开关已打开。
  </Step>
</Steps>

<Warning>如果后台应用刷新被禁用，您的小组件可能会显示过时的步数，只有在打开应用时才会更新。</Warning>
