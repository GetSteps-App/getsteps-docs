洞察为您提供步数和锻炼数据的年度回顾，包含统计数据、记录和有趣的比较。

## 步数分析

{/* TODO: Screenshot of Insights yearly receipt / steps analytics */}
<Frame caption="洞察年度收据，包含总步数、距离和卡路里">
  <img src="/images/screenshots/insights-yearly-receipt.jpg" alt="Insights yearly receipt" />

您的年度步数摘要包括：

- **年度收据** - 全年总步数、距离和卡路里
- **每日平均** - 您每天的平均步数
- **最佳日** - 您单日最高步数及日期
- **最佳月** - 总步数最多的月份
- **高峰时段** - 您最活跃的一天中的时段
- **工作日分析** - 按星期几比较步数的图表，包含工作日与周末平均值
- **达成目标次数** - 您达到步数目标的总天数
- **最佳连续** - 您最长的连续打卡天数

Steps 将您的年度总量转换为有趣的现实生活比较，涵盖距离、卡路里、节省的 CO2 等更多方面。看看您的步行与日常活动相比如何。

## 锻炼分析

如果您记录过锻炼，您还将看到：

- **锻炼航班票** - 总锻炼次数、锻炼时间、距离和卡路里
- **锻炼分析** - 最常见的锻炼类型、户外与室内次数、工作日与周末活动

## 个人记录

{/* TODO: Screenshot of personal records grid */}
<Frame caption="显示所有时间最佳成绩的个人记录网格">
  <img src="/images/screenshots/insights-personal-records.jpg" alt="Personal records" />
</Frame>

可滚动网格显示您的 8 项全时间个人记录：

| 记录 | 追踪内容 |
|--------|----------------|
| **最快** | 最佳配速 |
| **消耗最多** | 单次锻炼消耗最多卡路里 |
| **最早** | 最早锻炼开始时间 |
| **最大爬升** | 最大海拔爬升 |
| **最长** | 最长锻炼时长 |
| **最远** | 最长锻炼距离 |
| **最晚** | 最晚锻炼开始时间 |
| **最高心率** | 记录的最高心率 |

点击任意记录查看完整的锻炼详情。

## 年份选择器

使用工具栏中的年份选择器在不同年份之间切换，比较您随时间的活动变化。
