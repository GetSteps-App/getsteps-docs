除了[每日目标](/features/daily-goal)之外，您还可以为整年设置一个步数目标，并全年跟踪您实现它的进度节奏。

## 设置您的年度目标

{/* TODO: Screenshot of the yearly goal picker showing the daily equivalent */}
<Frame caption="年度目标选择器，显示每日相当步数">
  <img src="/images/screenshots/yearly-goal-picker.jpg" alt="Yearly goal picker" />
</Frame>

1. 打开**设置**
2. 点击**年度目标**
3. 使用滚轮选择器选择您的目标，范围为 **500,000 到 10,000,000 步**，步长为 **50,000**
4. 选择器会显示您的每日相当步数——例如*"大约每天 10,000 步"*——以便您选择一个切合实际的目标

默认年度目标为 **3,650,000 步**，大约相当于每天 10,000 步。

## 追踪您的进度

### 洞察卡片

{/* TODO: Screenshot of the yearly goal progress card in Insights */}
<Frame caption="洞察标签页中的年度目标进度卡片">
  <img src="/images/screenshots/yearly-goal-insights-card.jpg" alt="Yearly goal progress card" />
</Frame>

打开[洞察](/features/insights-overview)标签页即可查看您的年度目标卡片。它会显示：

- 您**迄今为止的步数**与您的目标对比（例如*3,650,000 步中的 2,400,000 步*）
- 带有配速标记的**进度条**，显示您今天应处于的位置以保持在计划进度上
- **状态**行——**按计划进行**、**按计划进行，领先 N 天**、**落后计划 N 天**，或**已于 MM/DD 达成目标**
- 在您仍在努力实现时：**完成所需的每日步数**和您的**预计年终总量**
- 一旦达成：您**超出目标**多少以及您**达成目标的日期**

对于已经结束的年份，卡片会显示您达到的**百分比**和该年的**每日平均值**。

<Tip>"按计划进行"会将您迄今为止的步数与均匀的每日节奏对比——即您的目标平摊到今年已过去的天数。保持在该线及以上，您就处于按时完成的节奏上。</Tip>

### 主页洞察

将主页洞察切换为**年度目标**模式，即可在[主页仪表盘](/features/home-dashboard)上步数下方查看您的年度进度。它会显示一条带有配速标记的细长进度条，以及两项关键统计数据——完成所需的每日步数和您的预计总量，或者在您达成目标后显示您的超出量和完成日期。

点击该洞察可循环切换模式，或在**设置 → 主页洞察**中选择它。

### 年度目标小组件

<Note>年度目标小组件是 Pro 功能。设置您的年度目标以及在洞察和主页洞察中查看它是免费的。</Note>

将**年度目标**小组件添加到您的主屏幕，即可一目了然地查看。它提供两种尺寸：

- **小号**——年份、迄今为止的步数、进度条和简短状态
- **中号**——同上，外加显示完成所需每日步数和您预计总量的统计磁贴

设置方法请参见[小组件](/features/widgets)。
