| Stile | Descrizione |
|-------|-------------|
| **Trasparente** | Le statistiche del tuo allenamento su sfondo trasparente, pronte per essere sovrapposte a qualsiasi immagine |
| **Sovrapposto** | Statistiche sovrapposte direttamente sulla mappa del percorso |
| **Cornice** | La tua mappa del percorso con una cornice con bordo pulito per le statistiche |
| **Foto** | Statistiche sovrapposte a una foto che scegli dalla tua libreria |

<Tip>Gli stili Sovrapposto e Cornice sono disponibili per gli allenamenti all'aperto con percorsi GPS.</Tip>

## Proporzioni

Seleziona la proporzione migliore per la destinazione di condivisione:

| Proporzione | Ideale per |
|-------------|-----------|
| **16:9** | YouTube, post in orizzontale |
| **4:3** | Presentazioni, foto in orizzontale |
| **1:1** | Feed Instagram, foto profilo |
| **3:4** | Foto in verticale |
| **9:16** | Instagram Stories, TikTok, Reels |

## Opzioni mappa

Per gli stili Sovrapposto e Cornice, personalizza l'aspetto della mappa:

- **Tipo di mappa** - Standard o Satellitare
- **Aspetto** - Chiaro o Scuro
- **Punti di interesse** - Includi tutti o Escludi tutti (solo mappa Standard)
- **Stile altitudine** - Piatto o Realistico (solo mappa Satellitare)

## Salva nelle foto

<Note>Il salvataggio nelle foto è una funzionalità Pro.</Note>

Tocca il pulsante salva per esportare l'immagine stilizzata del tuo allenamento come PNG direttamente nella tua libreria foto.

## Esportazione GPX

<Note>L'esportazione GPX è una funzionalità Pro.</Note>

Esporta il percorso del tuo allenamento come file `.gpx` da usare in altre app di fitness, strumenti di mappatura o pianificatori di percorsi. Il file GPX include il tuo tracciato GPS completo con i timestamp.
