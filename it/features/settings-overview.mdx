| Impostazione | Opzioni | Note |
|-------------|---------|------|
| **Aspetto** | Sistema, Chiaro, Scuro | Segui il dispositivo o imposta manualmente |
| **Tema colore** | 7 preimpostati + Personalizzato | Funzionalità Pro. Vedi [Temi colore](/features/color-themes) |
| **Sistema di unità** | Metrico, Imperiale | Influisce sulla visualizzazione della distanza (km o mi) |
| **La settimana inizia il** | Domenica, Lunedì | Cambia il calendario e le viste settimanali |
| **Visualizza passi per** | Giorno, Settimana, Mese | Settimana e Mese sono funzionalità Pro |
| **Obiettivo giornaliero** | 1.000 - 20.000 | Vedi [Obiettivo giornaliero](/features/daily-goal) |
| **Blocco app** | On, Off | Funzionalità Pro. Vedi [Blocco app](/features/app-lock) |
| **Notifiche** | On, Off | Funzionalità Pro. Vedi [Notifiche](/features/notifications) |
| **Insight home** | Off, Messaggio, Media | Cosa appare sotto il conteggio dei passi |
| **Mostra meteo** | On, Off | Mostra temperatura e umidità |
| **Icona app** | 17 icone | Vedi [Icona app](/features/app-icon-selector) |
| **Widget schermata Home** | Guida alla configurazione | Istruzioni per aggiungere widget |

## Lingua

Steps segue l'impostazione della lingua del dispositivo. Per cambiare la lingua dell'app:

1. Apri **Impostazioni** del tuo iPhone
2. Vai su **Steps** nell'elenco delle app
3. Cambia l'impostazione **Lingua**
