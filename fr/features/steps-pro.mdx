## Plans d'abonnement

Steps Pro est disponible en plans individuels ou familiaux :

| Plan | Facturation | Essai |
|------|-------------|-------|
| **Mensuel** | Facturé mensuellement | Essai gratuit de 7 jours |
| **Annuel** | Facturé annuellement (meilleur rapport qualité-prix) | Aucun |
| **À vie** | Achat unique | Aucun |

<Tip>Les plans familiaux vous permettent de partager votre abonnement Steps Pro avec jusqu'à 5 membres de la famille via Family Sharing d'Apple.</Tip>

## Essai gratuit

Le plan mensuel inclut un **essai gratuit de 7 jours**. Vous ne serez pas facturé avant la fin de l'essai, et vous pouvez annuler à tout moment pendant la période d'essai.

## Restaurer un achat

Si vous avez déjà acheté Steps Pro :

1. Ouvrez l'écran Steps Pro
2. Appuyez sur **Restaurer l'achat** en bas
3. Votre abonnement sera restauré automatiquement

## Utiliser un code

Si vous disposez d'un code promotionnel ou d'offre :

1. Ouvrez l'écran Steps Pro
2. Appuyez sur **Utiliser** pour saisir votre code
3. L'abonnement sera appliqué à votre compte

## Gérer votre abonnement

Pour modifier ou annuler votre abonnement, rendez-vous dans **Réglages > Gérer l'abonnement** dans l'application Steps. Cela ouvre votre page de gestion des abonnements Apple.
