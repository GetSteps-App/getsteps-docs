Steps envoie des notifications pour célébrer votre progression et vous garder motivé tout au long de la journée.

<Note>Les notifications sont une fonctionnalité Pro.</Note>

### Objectif atteint

Déclenchée lorsque votre nombre de pas atteint votre objectif quotidien. Vous verrez un message de célébration avec votre réussite. Envoyée au maximum une fois par jour.

### Étape importante

Déclenchée lorsque vous atteignez **50 % de votre objectif quotidien**. Affiche votre pourcentage de progression et le nombre de pas restants pour atteindre votre cible. Envoyée au maximum une fois par jour.

### Fin d'entraînement

Envoyée après avoir terminé une session d'entraînement. Inclut la durée de votre entraînement et la distance parcourue. Le message varie en fonction du type d'entraînement (marche, course ou cyclisme).

### Applications déverrouillées

Si vous utilisez le [Verrouillage d'application](/features/app-lock), vous recevrez une notification lorsque votre objectif de pas est atteint et que vos applications verrouillées sont débloquées. Envoyée au maximum une fois par jour.

## Activer les notifications

1. Ouvrez les **Réglages** dans Steps
2. Activez les **Notifications**
3. Autorisez les notifications lorsque iOS vous le demande

## Désactiver les notifications

Désactivez les **Notifications** dans les Réglages de Steps pour arrêter toutes les notifications. Vous pouvez également gérer les notifications de Steps dans les **Réglages > Notifications > Steps** de votre iPhone.
