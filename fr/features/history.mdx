L'onglet Historique vous permet de parcourir vos enregistrements de pas quotidiens dans le temps à l'aide d'une vue calendrier ou d'une liste chronologique.

## Vue calendrier

Le calendrier affiche chaque jour avec un indicateur codé par couleur montrant votre progression :

- **Objectif atteint** - Mis en évidence dans votre couleur de thème « complété »
- **Progression partielle** - Teinte plus claire selon le pourcentage complété
- **Aucune donnée** - Pas de mise en évidence

Naviguez entre les mois à l'aide des boutons fléchés en haut du calendrier.

Passez en vue liste pour voir vos enregistrements quotidiens dans une liste défilante, affichant :

- Date
- Nombre de pas
- Distance
- Si vous avez atteint votre objectif

## Navigation par mois

Utilisez les contrôles de navigation mensuelle pour passer d'un mois à l'autre. Chaque mois affiche un résumé de votre activité pour cette période.

## Limites de l'historique

| Plan | Accès à l'historique |
|------|---------------------|
| **Gratuit** | 3 derniers mois |
| **Pro** | Historique illimité |

<Note>Passez à Steps Pro pour parcourir votre historique de pas complet au-delà des 3 derniers mois.</Note>

## Modes d'affichage

Vous pouvez modifier l'organisation de l'historique dans **Réglages > Afficher les pas par** :

- **Jour** - Enregistrements quotidiens individuels (gratuit)
- **Semaine** - Résumés hebdomadaires (Pro)
- **Mois** - Résumés mensuels (Pro)
