---
title: "Verrouillage d'application"
description: "Bloquez les applications distrayantes jusqu'à ce que vous atteigniez votre objectif quotidien de pas"
---

Le verrouillage d'application utilise Screen Time d'Apple pour bloquer les applications sélectionnées jusqu'à ce que vous atteigniez votre objectif quotidien de pas, vous aidant à rester actif avant de passer du temps devant l'écran.

<Note>Le verrouillage d'application est une fonctionnalité Pro.</Note>

<Steps>
  <Step title="Activer le verrouillage d'application">
    Allez dans **Réglages > Verrouillage d'application** et activez-le. Steps demandera l'autorisation Screen Time.
  </Step>
  <Step title="Choisir les applications à verrouiller">
    Sélectionnez les applications et catégories d'applications que vous souhaitez bloquer à l'aide du sélecteur d'applications d'Apple. Les réseaux sociaux, les jeux et les applications de divertissement sont des choix populaires.
  </Step>
  <Step title="Marcher pour déverrouiller">
    Les applications verrouillées sont bloquées par un bouclier jusqu'à ce que vous atteigniez votre objectif quotidien de pas. Une fois que vous avez atteint votre cible, toutes les applications sélectionnées se déverrouillent automatiquement.
  </Step>
</Steps>

## Réinitialisation quotidienne

Le verrouillage d'application se réinitialise à **minuit** chaque jour. Cela signifie :

- Les boucliers sont réappliqués au début de chaque nouvelle journée
- Une fois les applications déverrouillées, elles restent déverrouillées jusqu'à minuit — même si vous changez votre objectif
- Vous repartez à zéro chaque matin

## Messages de progression

Pendant que les applications sont verrouillées, vous verrez des messages de progression vous encourageant à continuer de marcher. Lorsque vous atteignez votre objectif, une notification **« Applications déverrouillées ! »** vous en informe.

## Changements d'objectif en cours de journée

- Si vous **abaissez** votre objectif et que vos pas actuels le dépassent déjà, les applications se déverrouillent immédiatement
- Si vous **relevez** votre objectif, les applications restent verrouillées jusqu'à ce que vous atteigniez la nouvelle cible

<Warning>Le verrouillage d'application nécessite l'autorisation Screen Time. Si vous refusez la permission, vous devrez l'accorder dans les réglages Screen Time de votre appareil pour utiliser cette fonctionnalité.</Warning>
