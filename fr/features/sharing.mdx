| Style | Description |
|-------|-------------|
| **Transparent** | Vos statistiques d'entraînement sur un fond transparent, prêtes à superposer sur n'importe quelle image |
| **Superposition** | Statistiques superposées directement sur votre carte d'itinéraire |
| **Cadre** | Votre carte d'itinéraire avec un cadre de statistiques bordé et propre |
| **Photo** | Statistiques superposées sur une photo que vous choisissez dans votre bibliothèque |

<Tip>Les styles Superposition et Cadre sont disponibles pour les entraînements extérieurs avec itinéraires GPS.</Tip>

## Formats d'image

Sélectionnez le meilleur format pour votre destination de partage :

| Format | Idéal pour |
|--------|------------|
| **16:9** | YouTube, publications en paysage |
| **4:3** | Présentations, photos en paysage |
| **1:1** | Fil Instagram, photos de profil |
| **3:4** | Photos en portrait |
| **9:16** | Stories Instagram, TikTok, Reels |

## Options de carte

Pour les styles Superposition et Cadre, personnalisez l'apparence de la carte :

- **Type de carte** - Standard ou Satellite
- **Apparence** - Clair ou Sombre
- **Points d'intérêt** - Inclure tout ou Exclure tout (carte Standard uniquement)
- **Style d'altitude** - Plat ou Réaliste (carte Satellite uniquement)

## Enregistrer dans les Photos

<Note>L'enregistrement dans les Photos est une fonctionnalité Pro.</Note>

Appuyez sur le bouton d'enregistrement pour exporter votre image d'entraînement stylisée en tant que fichier PNG directement dans votre photothèque.

## Export GPX

<Note>L'export GPX est une fonctionnalité Pro.</Note>

Exportez votre itinéraire d'entraînement en tant que fichier `.gpx` pour l'utiliser dans d'autres applications de fitness, outils de cartographie ou planificateurs d'itinéraires. Le fichier GPX inclut votre tracé GPS complet avec les horodatages.
