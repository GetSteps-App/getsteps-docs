La vue Détail de l'activité vous offre une analyse complète de tout entraînement enregistré, incluant votre itinéraire, vos statistiques et vos options de partage.

Pour les entraînements extérieurs, une carte affiche votre itinéraire GPS tracé dans la couleur de l'entraînement. Vous pouvez basculer entre :

- Vue de carte **Standard**
- Vue par **Satellite**

## Statistiques d'entraînement

Chaque vue détaillée d'entraînement affiche :

- **Durée** - Temps total d'entraînement
- **Distance** - Distance totale parcourue
- **Allure** - Allure moyenne (pour la marche et la course)
- **Dénivelé positif** - Montée totale pendant l'entraînement
- **Fréquence cardiaque** - Fréquence cardiaque moyenne et maximale
- **Calories** - Énergie active brûlée
- **Météo** - Température et humidité pendant l'entraînement

## Lecture de l'itinéraire

<Note>La lecture de l'itinéraire est une fonctionnalité Pro.</Note>

Rejouez votre itinéraire d'entraînement sur la carte pour voir comment vous vous êtes déplacé pendant votre session. La lecture anime votre position le long de l'itinéraire pour revivre l'entraînement.

| Style | Description |
|-------|-------------|
| **Transparent** | Statistiques sur un fond transparent |
| **Superposition** | Statistiques superposées sur votre carte d'itinéraire |
| **Cadre** | Carte de l'itinéraire avec un cadre de statistiques bordé |
| **Photo** | Statistiques superposées sur une photo de votre bibliothèque |

Vous pouvez également choisir parmi cinq formats d'image : 16:9, 4:3, 1:1, 3:4 et 9:16.

Pour les styles basés sur la carte, des options supplémentaires incluent l'apparence de la carte (clair/sombre), la visibilité des points d'intérêt et le style d'altitude.

Voir [Partage](/features/sharing) pour tous les détails.

## Export GPX

<Note>L'export GPX est une fonctionnalité Pro.</Note>

Exportez votre itinéraire d'entraînement en tant que fichier `.gpx` pour l'utiliser dans d'autres applications de fitness ou outils de cartographie. Appuyez sur le bouton de partage et sélectionnez l'export GPX pour enregistrer ou envoyer le fichier.
