| Réglage | Options | Notes |
|---------|---------|-------|
| **Apparence** | Système, Clair, Sombre | Suivez votre appareil ou définissez manuellement |
| **Thème de couleur** | 7 thèmes prédéfinis + Personnalisé | Fonctionnalité Pro. Voir [Thèmes de couleur](/features/color-themes) |
| **Système d'unités** | Métrique, Impérial | Affecte l'affichage des distances (km ou miles) |
| **La semaine commence le** | Dimanche, Lundi | Modifie le calendrier et les vues hebdomadaires |
| **Afficher les pas par** | Jour, Semaine, Mois | Semaine et Mois sont des fonctionnalités Pro |
| **Objectif quotidien** | 1 000 - 20 000 | Voir [Objectif quotidien](/features/daily-goal) |
| **Verrouillage d'application** | Activé, Désactivé | Fonctionnalité Pro. Voir [Verrouillage d'application](/features/app-lock) |
| **Notifications** | Activées, Désactivées | Fonctionnalité Pro. Voir [Notifications](/features/notifications) |
| **Aperçu de l'accueil** | Désactivé, Message, Moyenne | Ce qui s'affiche sous votre nombre de pas |
| **Afficher la météo** | Activé, Désactivé | Afficher la température et l'humidité |
| **Icône d'application** | 17 icônes | Voir [Icône d'application](/features/app-icon-selector) |
| **Widgets d'écran d'accueil** | Guide de configuration | Instructions pour ajouter des widgets |

## Langue

Steps suit le paramètre de langue de votre appareil. Pour changer la langue de l'application :

1. Ouvrez les **Réglages** de votre iPhone
2. Allez dans **Steps** dans la liste des applications
3. Modifiez le paramètre **Langue**
