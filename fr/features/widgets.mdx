Steps propose une variété de widgets pour que vous puissiez voir votre nombre de pas, vos séries et vos données d'entraînement sans ouvrir l'application.

## Live Activity

Pendant un entraînement actif, une Live Activity apparaît sur votre écran de verrouillage et dans la Dynamic Island (iOS 26+). Elle affiche :

- Icône du type d'entraînement
- Temps écoulé
- Distance parcourue
- Calories brûlées
- Allure actuelle

La Dynamic Island alterne entre les statistiques toutes les 30 secondes.

## Ajouter des widgets

<Steps>
  <Step title="Appuyez longuement sur votre écran d'accueil">
    Touchez et maintenez une zone vide de votre écran d'accueil jusqu'à ce que les applications commencent à trembler.
  </Step>
  <Step title="Appuyez sur le bouton +">
    Appuyez sur le bouton **+** dans le coin supérieur gauche pour ouvrir la galerie de widgets.
  </Step>
  <Step title="Recherchez Steps">
    Tapez « Steps » dans la barre de recherche pour trouver tous les widgets disponibles.
  </Step>
  <Step title="Choisissez un widget">
    Sélectionnez le type et la taille de widget souhaités, puis appuyez sur **Ajouter un widget**.
  </Step>
</Steps>

Le widget Pas du jour est également disponible sur votre écran de verrouillage en trois styles :

- **Rectangulaire** - Affiche le nombre de pas et une barre de progression
- **En ligne** - Nombre de pas compact dans la ligne de date
- **Circulaire** - Nombre de pas dans une jauge ronde

<Tip>Vos widgets se mettent automatiquement à jour avec vos dernières données de pas et reflètent votre thème de couleur choisi.</Tip>
