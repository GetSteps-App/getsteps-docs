設定画面では、Stepsの見た目や動作をカスタマイズできます。ホームダッシュボードの歯車アイコンからアクセスできます。

| 設定 | オプション | 備考 |
|------|----------|------|
| **外観** | システム、ライト、ダーク | デバイスに従うか手動で設定 |
| **カラーテーマ** | 7つのプリセット + カスタム | Pro機能。[カラーテーマ](/ja/features/color-themes)を参照 |
| **単位系** | メトリック、インペリアル | 距離表示に影響（kmまたはmi） |
| **週の開始日** | 日曜日、月曜日 | カレンダーと週間表示が変更されます |
| **歩数の表示** | 日、週、月 | 週と月はPro機能 |
| **1日の目標** | 1,000 - 20,000 | [1日の目標](/ja/features/daily-goal)を参照 |
| **アプリロック** | オン、オフ | Pro機能。[アプリロック](/ja/features/app-lock)を参照 |
| **通知** | オン、オフ | Pro機能。[通知](/ja/features/notifications)を参照 |
| **ホームインサイト** | オフ、メッセージ、平均 | 歩数の下に表示される内容 |
| **天気を表示** | オン、オフ | 気温と湿度を表示 |
| **アプリアイコン** | 17種類のアイコン | [アプリアイコン](/ja/features/app-icon-selector)を参照 |
| **ホーム画面ウィジェット** | 設定ガイド | ウィジェットの追加手順 |

## 言語

StepsはデバイスのLanguage設定に従います。アプリの言語を変更するには：

1. iPhoneの**設定**を開く
2. アプリ一覧から**Steps**に移動
3. **言語**設定を変更

## サポート

- **フィードバックボード** - 機能リクエストの送信とアイデアへの投票
- **レビューを書く** - App StoreでStepsを評価

- **Apple Healthデータの再同期** - アクティビティデータを更新

- **フィットネス計算ツール** - [getsteps.app/tools](https://getsteps.app/tools)
