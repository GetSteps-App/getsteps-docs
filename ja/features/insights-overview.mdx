インサイトでは、歩数とワークアウトデータの年間レビューを、統計、記録、楽しい比較とともに確認できます。

## 歩数分析

年間歩数サマリーには以下が含まれます：

- **年間レシート** - 年間の合計歩数、距離、カロリー
- **1日の平均** - 1日あたりの平均歩数
- **ベストデイ** - 1日の最高歩数とその日付
- **ベストマンス** - 合計歩数が最も多かった月
- **ピークアワー** - 最も活動的な時間帯
- **曜日別内訳** - 曜日別の歩数比較チャート、平日と週末の平均値
- **目標達成日数** - 歩数目標を達成した合計日数
- **ベストストリーク** - 最長の連続達成日数

## ワークアウト分析

ワークアウトを記録している場合、以下も表示されます：

- **ワークアウトフライトチケット** - ワークアウト合計回数、運動時間、距離、カロリー
- **ワークアウト内訳** - 最も多いワークアウトタイプ、屋外vs屋内の回数、平日vs週末のアクティビティ

## パーソナルレコード

<Frame caption="オールタイムベストを表示するパーソナルレコードグリッド">
  <img src="/images/screenshots/insights-personal-records.jpg" alt="パーソナルレコード" />
</Frame>

スクロール可能なグリッドに8つのオールタイムパーソナルレコードが表示されます：

| レコード | 記録内容 |
|----------|----------|
| **最速** | ベストペース |
| **最多消費** | 1回のワークアウトでの最多カロリー消費 |
| **最早** | 最も早いワークアウト開始時刻 |
| **最多上昇** | 最大の獲得標高 |
| **最長** | 最長のワークアウト時間 |
| **最遠** | 最大のワークアウト距離 |
| **最遅** | 最も遅いワークアウト開始時刻 |
| **最大心拍数** | 記録された最高心拍数 |

レコードをタップすると、ワークアウトの詳細を確認できます。

## 年ピッカー

ツールバーの年ピッカーを使って年を切り替え、アクティビティを比較できます。
