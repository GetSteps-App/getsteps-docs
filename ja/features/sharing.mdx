Stepsでは、ワークアウトの成果を美しくスタイリングされた画像として共有したり、ルートデータをGPXファイルとしてエクスポートしたりできます。

| スタイル | 説明 |
|---------|------|
| **透過** | ワークアウトの統計情報を透明な背景に表示。任意の画像に重ねて使用可能 |
| **オーバーレイ** | ルートマップ上に統計情報を直接重ねて表示 |
| **フレーム** | ルートマップにすっきりとした枠付きの統計情報フレームを表示 |
| **フォト** | ライブラリから選んだ写真の上に統計情報を重ねて表示 |

<Tip>オーバーレイとフレームスタイルは、GPSルート付きの屋外ワークアウトで利用可能です。</Tip>

## アスペクト比

共有先に最適な比率を選択できます：

| 比率 | 最適な用途 |
|------|----------|
| **16:9** | YouTube、横長投稿 |
| **4:3** | プレゼンテーション、横長写真 |
| **1:1** | Instagramフィード、プロフィール写真 |
| **3:4** | 縦長写真 |
| **9:16** | Instagramストーリーズ、TikTok、リール |

## マップオプション

オーバーレイとフレームスタイルでは、マップの外観をカスタマイズできます：

- **マップタイプ** - 標準またはサテライト
- **外観** - ライトまたはダーク
- **ポイントオブインタレスト** - すべて表示またはすべて非表示（標準マップのみ）
- **標高スタイル** - フラットまたはリアリスティック（サテライトマップのみ）

## 写真に保存

<Note>写真への保存はPro機能です。</Note>

保存ボタンをタップすると、スタイリングされたワークアウト画像がPNGとしてフォトライブラリに直接エクスポートされます。

## GPXエクスポート

<Note>GPXエクスポートはPro機能です。</Note>

ワークアウトのルートを`.gpx`ファイルとしてエクスポートし、他のフィットネスアプリ、マッピングツール、ルートプランナーで使用できます。GPXファイルにはタイムスタンプ付きの完全なGPSトラックが含まれます。
