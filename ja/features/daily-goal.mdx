毎日の目標は、毎日達成を目指す歩数ターゲットです。目標を達成するとストリークが積み上がり、マイルストーンメッセージがアンロックされます。

## 目標の設定

1. **設定**を開く
2. **毎日の目標**をタップ
3. スクロールピッカーを使って**1,000歩から20,000歩**の間を500歩刻みで選択
4. 単位設定（kmまたはmi）に基づく推定距離がピッカーの下に表示されます

## マイルストーンティア

目標を調整すると、難易度レベルを示すマイルストーンメッセージが表示されます：

| 目標範囲 | マイルストーン | アイコン |
|----------|---------------|----------|
| 1,000 - 2,999 | はじめの一歩 | Leaf |
| 3,000 - 4,999 | 勢いをつけよう | Bolt |
| 5,000 - 7,999 | 良い毎日の習慣 | Star |
| 8,000 - 9,999 | アクティブなライフスタイル | Paperplane |
| 10,000 - 12,999 | 素晴らしい目標 | Target |
| 13,000 - 15,999 | ハイアチーバー | Trophy |
| 16,000 - 20,000 | アスリートレベル | Medal |

<Tip>まずは確実に達成できる目標から始めて、徐々に上げていきましょう。維持できない高い目標を設定するよりも、ストリークを積み重ねることが大切です。</Tip>

## 目標とストリーク

毎日の目標は[ストリーク](/ja/features/streaks)に直接影響します。目標を達成した日ごとに連続ストリークのカウントが増えます。目標未達成の場合、ストリークはリセットされます。

日中に目標を変更した場合：

- **引き下げ**た場合：現在の歩数がすでに新しい目標を超えていれば、その日は達成としてカウントされます
- **引き上げ**た場合：その日がカウントされるには、新しい高い目標を達成する必要があります

## 目標の進捗

毎日の目標に対する進捗は、Steps内のさまざまな場所で確認できます：

- **ホームダッシュボード** - 歩数カードの周りのプログレスリング
- **ウィジェット** - 歩数ウィジェットに進捗を表示
- **Watchコンプリケーション** - Apple Watchに目標の進捗を表示
- **アプリロック** - 目標達成時にブロックされたアプリがアンロック
