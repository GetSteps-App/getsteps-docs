## Icon ändern

1. Öffne **Einstellungen**
2. Tippe auf **App-Icon**
3. Durchsuche das Icon-Raster und tippe auf deine Wahl
4. iOS bestätigt die Icon-Änderung

<Tip>Jedes Icon hat eine dunkle Variante, die zum Dunkelmodus deines Systems passt. Das Icon passt sich automatisch den Darstellungseinstellungen deines Geräts an.</Tip>
