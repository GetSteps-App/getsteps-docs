| Einstellung | Optionen | Hinweise |
|-------------|----------|----------|
| **Darstellung** | System, Hell, Dunkel | Folgt deinem Gerät oder manuell einstellen |
| **Farbthema** | 7 Voreinstellungen + Benutzerdefiniert | Pro-Funktion. Siehe [Farbthemen](/features/color-themes) |
| **Einheitensystem** | Metrisch, Imperial | Beeinflusst die Distanzanzeige (km oder Meilen) |
| **Woche beginnt am** | Sonntag, Montag | Ändert Kalender- und Wochenansichten |
| **Schritte anzeigen nach** | Tag, Woche, Monat | Woche und Monat sind Pro-Funktionen |
| **Tagesziel** | 1.000 - 20.000 | Siehe [Tagesziel](/features/daily-goal) |
| **App-Sperre** | Ein, Aus | Pro-Funktion. Siehe [App-Sperre](/features/app-lock) |
| **Benachrichtigungen** | Ein, Aus | Pro-Funktion. Siehe [Benachrichtigungen](/features/notifications) |
| **Startseiten-Einblick** | Aus, Nachricht, Durchschnitt | Was unter deiner Schrittzahl angezeigt wird |
| **Wetter anzeigen** | Ein, Aus | Temperatur und Luftfeuchtigkeit anzeigen |
| **App-Icon** | 17 Icons | Siehe [App-Icon](/features/app-icon-selector) |
| **Startbildschirm-Widgets** | Einrichtungsanleitung | Anweisungen zum Hinzufügen von Widgets |

## Sprache

Steps folgt der Spracheinstellung deines Geräts. So änderst du die App-Sprache:

1. Öffne die **Einstellungen** deines iPhones
2. Gehe zu **Steps** in der App-Liste
3. Ändere die **Sprache**-Einstellung
