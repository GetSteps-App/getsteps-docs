Steps는 Apple 건강에서 다음 데이터를 읽습니다:

- **걸음 수** - 일일 걸음 수
- **걷기 + 달리기 거리** - 도보로 이동한 거리
- **활성 에너지 소모** - 활동으로 소모된 칼로리
- **오른 층수** - 올라간 층수
- **심박수** - 운동 중 사용
- **운동 경로** - 기록된 운동의 GPS 경로

<Tip>Steps는 Apple 건강 데이터를 쓰거나 수정하지 않습니다. 기존 활동 정보만 읽습니다.</Tip>

## 개인정보 보호

건강 데이터는 기기에 저장됩니다. Steps는 피트니스 데이터를 어떤 서버에도 업로드하지 않습니다. 모든 처리는 iPhone에서 로컬로 이루어집니다.

## 건강 데이터 재동기화

걸음 수가 정확하지 않거나 데이터가 누락된 것 같으면 Apple 건강에서 재동기화할 수 있습니다:

1. Steps에서 **설정**을 여세요
2. 지원 섹션으로 스크롤하세요
3. **Apple 건강 데이터 재동기화**를 탭하세요

이렇게 하면 Apple 건강의 모든 활동 데이터가 새로고침됩니다.

## 온보딩 다시 시작

설정 과정을 다시 진행하려면:

1. Steps에서 **설정**을 여세요
2. **온보딩 다시 시작**을 탭하세요
3. 확인 메시지가 나타나면 확인하세요

이를 통해 일일 목표와 권한을 다시 구성할 수 있습니다.
