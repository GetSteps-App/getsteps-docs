| 설정 | 옵션 | 참고 |
|------|------|------|
| **외관** | 시스템, 밝게, 어둡게 | 기기를 따르거나 수동으로 설정 |
| **색상 테마** | 7가지 프리셋 + 맞춤 | Pro 기능. [색상 테마](/ko/features/color-themes) 참조 |
| **단위 시스템** | 미터법, 야드파운드법 | 거리 표시에 영향 (km 또는 mi) |
| **주 시작일** | 일요일, 월요일 | 캘린더 및 주간 보기 변경 |
| **걸음 수 보기 기준** | 일간, 주간, 월간 | 주간 및 월간은 Pro 기능 |
| **일일 목표** | 1,000 - 20,000 | [일일 목표](/ko/features/daily-goal) 참조 |
| **앱 잠금** | 켜기, 끄기 | Pro 기능. [앱 잠금](/ko/features/app-lock) 참조 |
| **알림** | 켜기, 끄기 | Pro 기능. [알림](/ko/features/notifications) 참조 |
| **홈 인사이트** | 끄기, 메시지, 평균 | 걸음 수 아래에 표시되는 내용 |
| **날씨 표시** | 켜기, 끄기 | 기온 및 습도 표시 |
| **앱 아이콘** | 17가지 아이콘 | [앱 아이콘](/ko/features/app-icon-selector) 참조 |
| **홈 화면 위젯** | 설정 가이드 | 위젯 추가 안내 |

## 언어

Steps는 기기의 언어 설정을 따릅니다. 앱 언어를 변경하려면:

1. iPhone의 **설정**을 여세요
2. 앱 목록에서 **Steps**로 이동하세요
3. **언어** 설정을 변경하세요
