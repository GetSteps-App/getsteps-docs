| 스타일 | 설명 |
|--------|------|
| **투명** | 투명 배경의 운동 통계, 어떤 이미지 위에든 겹칠 준비 완료 |
| **오버레이** | 경로 지도 위에 직접 겹친 통계 |
| **프레임** | 깔끔한 테두리가 있는 통계 프레임과 경로 지도 |
| **사진** | 라이브러리에서 선택한 사진 위의 통계 |

<Tip>오버레이 및 프레임 스타일은 GPS 경로가 있는 실외 운동에서 사용할 수 있습니다.</Tip>

## 화면 비율

공유 대상에 맞는 최적의 비율을 선택하세요:

| 비율 | 추천 용도 |
|------|----------|
| **16:9** | YouTube, 가로 게시물 |
| **4:3** | 프레젠테이션, 가로 사진 |
| **1:1** | Instagram 피드, 프로필 사진 |
| **3:4** | 세로 사진 |
| **9:16** | Instagram 스토리, TikTok, 릴스 |

## 지도 옵션

오버레이 및 프레임 스타일에서는 지도 외관을 맞춤 설정할 수 있습니다:

- **지도 유형** - 표준 또는 위성
- **외관** - 밝게 또는 어둡게
- **관심 지점** - 모두 포함 또는 모두 제외 (표준 지도에서만)
- **고도 스타일** - 평면 또는 사실적 (위성 지도에서만)

## 사진에 저장

<Note>사진에 저장은 Pro 기능입니다.</Note>

저장 버튼을 탭하여 스타일이 적용된 운동 이미지를 PNG로 사진 라이브러리에 직접 내보내세요.

## GPX 내보내기

<Note>GPX 내보내기는 Pro 기능입니다.</Note>

운동 경로를 `.gpx` 파일로 내보내어 다른 피트니스 앱, 매핑 도구, 경로 플래너에서 사용하세요. GPX 파일에는 타임스탬프가 포함된 완전한 GPS 트랙이 포함됩니다.
