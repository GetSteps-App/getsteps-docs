The Settings screen lets you customize how Steps looks and works. Access it from the gear icon on the Home Dashboard.

## App Settings

| Setting | Options | Notes |
|---------|---------|-------|
| **Appearance** | System, Light, Dark | Follow your device or set manually |
| **Color Theme** | 7 presets + Custom | Pro feature. See [Color Themes](/docs/features/color-themes) |
| **Unit System** | Metric, Imperial | Affects distance display (km or mi) |
| **Week Starts On** | Sunday, Monday | Changes calendar and weekly views |
| **View Steps By** | Day, Week, Month | Week and Month are Pro features |
| **Daily Goal** | 1,000 - 20,000 | See [Daily Goal](/docs/features/daily-goal) |
| **App Lock** | On, Off | Pro feature. See [App Lock](/docs/features/app-lock) |
| **Notifications** | On, Off | Pro feature. See [Notifications](/docs/features/notifications) |
| **Home Insight** | Off, Message, Average | What shows below your step count |
| **Show Weather** | On, Off | Display temperature and humidity |
| **App Icon** | 17 icons | See [App Icon](/docs/features/app-icon-selector) |
| **Home Screen Widgets** | Setup guide | Instructions for adding widgets |

## Language

Steps follows your device's language setting. To change the app language:

1. Open your iPhone's **Settings**
2. Go to **Steps** in the app list
3. Change the **Language** setting
