| Стиль | Описание |
|-------|----------|
| **Прозрачный** | Статистика тренировки на прозрачном фоне, готовая для наложения на любое изображение |
| **Наложение** | Статистика, наложенная на карту маршрута |
| **Рамка** | Карта маршрута с аккуратной рамкой статистики |
| **Фото** | Статистика, наложенная на выбранную вами фотографию из библиотеки |

<Tip>Стили «Наложение» и «Рамка» требуют тренировки с GPS-маршрутом. Для тренировок без маршрута доступны только «Прозрачный» и «Фото».</Tip>

## Соотношения сторон

Выберите оптимальное соотношение для платформы публикации:

| Соотношение | Лучше всего для |
|-------------|-----------------|
| **16:9** | YouTube, горизонтальные публикации |
| **4:3** | Презентации, горизонтальные фото |
| **1:1** | Лента Instagram, аватары |
| **3:4** | Вертикальные фото |
| **9:16** | Instagram Stories, TikTok, Reels |

## Настройки карты

Для стилей «Наложение» и «Рамка» можно настроить внешний вид карты:

- **Тип карты** — Стандартная или Спутниковая
- **Оформление** — Светлое или Тёмное
- **Точки интереса** — Показать все или Скрыть все (только для стандартной карты)
- **Стиль рельефа** — Плоский или Реалистичный (только для спутниковой карты)

## Сохранение в Фото

<Note>Сохранение в Фото — функция Pro.</Note>

Нажмите кнопку сохранения, чтобы экспортировать стилизованное изображение тренировки в формате PNG прямо в вашу фотобиблиотеку.

## Экспорт GPX

<Note>Экспорт GPX — функция Pro.</Note>

Экспортируйте маршрут тренировки в виде файла `.gpx` для использования в других фитнес-приложениях, картографических инструментах или планировщиках маршрутов. Файл GPX содержит полный GPS-трек с метками времени.

## Поделиться шагами

Помимо тренировок, вы можете поделиться количеством шагов прямо с главного экрана. Нажмите кнопку «Поделиться» в верхней панели инструментов, чтобы открыть предпросмотр публикации шагов.

| Стиль | Описание |
|-------|----------|
| **Фото** | Ваше количество шагов и основные показатели поверх фотографии из вашей библиотеки |
| **Прозрачный** | Аккуратный счётчик шагов на прозрачном фоне, готовый для наложения на собственный фон |

Опубликованное изображение включает количество шагов за выбранный период вместе с показателями, такими как расстояние, калории и пройденные этажи.

- В режиме **Фото** выберите соотношение сторон (Квадрат, 16:9, 4:3, 3:4 или 9:16) и выберите фотографию.
- В режиме **Прозрачный** выберите светлое или тёмное оформление.

<Note>Сохранение изображения шагов в Фото — функция Pro.</Note>
