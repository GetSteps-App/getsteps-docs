Цветовые темы позволяют настроить внешний вид Steps, изменяя цвета кольца прогресса, графиков и виджетов.

<Note>Цветовые темы — функция Pro. Бесплатные пользователи используют тему «Классическая» по умолчанию.</Note>

Steps включает 7 встроенных цветовых тем. Каждая тема имеет два цвета: один для процесса выполнения и один для достижения цели.

| Тема | В процессе | Выполнено |
|------|------------|-----------|
| **Классическая** | Оранжевый | Зелёный |
| **Сланцевая** | Сланцево-синий | Синий |
| **Коралловая** | Коралловый | Розовый |
| **Лесная** | Янтарный | Изумрудный |
| **Сумеречная** | Фиолетовый | Индиго |
| **Бирюзовая** | Бирюзовый | Голубой |
| **Румяная** | Розовый | Пурпурный |

## Пользовательские цвета

Выберите собственные цвета с помощью пользовательской темы. Вы можете задать отдельные цвета для:

- **В процессе** — цвет, отображаемый пока вы идёте к цели
- **Выполнено** — цвет, отображаемый после достижения цели

Используйте палитру цветов iOS для выбора любого цвета.

## Где применяются темы

Выбранные цвета темы отображаются по всему приложению:

- Кольцо прогресса на главном экране
- Столбцы и выделения на графиках
- Виджеты на домашнем экране и экране блокировки
- Акценты значков серий

<Tip>Смена темы автоматически обновляет ваши виджеты.</Tip>

## Как сменить тему

1. Откройте **Настройки**
2. Нажмите **Цветовая тема**
3. Выберите готовую тему или нажмите **Пользовательская**, чтобы выбрать свои цвета
