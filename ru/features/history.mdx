Вкладка «История» позволяет просматривать ежедневные записи о шагах за прошедшее время в виде календаря или хронологического списка.

Календарь отображает каждый день с цветовым индикатором вашего прогресса:

- **Цель достигнута** — выделено цветом вашей темы завершения
- **Частичный прогресс** — более светлый оттенок в зависимости от процента выполнения
- **Нет данных** — без выделения

Переключайтесь между месяцами с помощью стрелок в верхней части календаря.

Переключитесь на вид списка, чтобы увидеть ежедневные записи в прокручиваемом списке:

- Дата
- Количество шагов
- Расстояние
- Достигнута ли цель

## Навигация по месяцам

Используйте элементы навигации по месяцам для переключения между месяцами. Для каждого месяца отображается сводка вашей активности за этот период.

## Ограничения истории

| План | Доступ к истории |
|------|-----------------|
| **Бесплатный** | Последние 3 месяца |
| **Pro** | Неограниченная история |

<Note>Перейдите на Steps Pro, чтобы просматривать полную историю шагов за период более 3 месяцев.</Note>

## Режимы отображения

Вы можете изменить способ группировки истории в **Настройки > Просмотр шагов по**:

- **День** — ежедневные записи (бесплатно)
- **Неделя** — недельные сводки (Pro)
- **Месяц** — месячные сводки (Pro)
